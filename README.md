# Interactive Excel Dashboard — End to End Project

**Author:** Shruti Chavan  
**Tool:** Microsoft Excel (.xlsx)  
**Data:** Simulated sample dataset for the tutorial project  
**Status:** Portfolio project

## About
This repository contains a complete interactive Excel dashboard built end-to-end.It includes raw → cleaned sample data, data processing steps, PivotTables and chart visualizations, and a wide-screen interactive dashboard designed for screen presentation. The project demonstrates practical Excel skills for business analysis: data cleaning, intermediate formulas, pivot-based reporting and dashboard design.

## Project Summary
Built a full workflow from raw dataset to polished dashboard:
- Data cleaning & transformation steps (sheet / Power Query equivalents)  
- Analytical tables / PivotTables for aggregations  
- Interactive visuals and KPI tiles for quick insights  
- Dashboard layout optimized for presentation and screen viewing

## Files
